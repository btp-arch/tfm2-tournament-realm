# Roadmap

## Milestone 0: Project foundation

- Next.js + TypeScript app shell
- Placeholder pages
- Supabase helpers
- Initial SQL schema and RLS policies
- Setup documentation and project guardrails

## Milestone 1: Authentication and profiles

- Supabase Auth wiring
- Profile creation and editing
- Role-aware navigation
- Generated database types in normal workflow

## Milestone 2: Tournament registration

- Tournament listing and detail views
- Organizer tournament creation/editing
- Free-entry registration and withdrawal
- Organizer registration management

## Milestone 3: Match rooms

- Match assignment views
- Check-in flow
- Random host assignment
- Host side choice tracking
- BO1/BO3/BO5 display support

## Milestone 4: Reporting and disputes

- Result reports
- Screenshot/evidence upload
- Opponent confirmation
- Dispute opening and organizer/admin resolution

## Milestone 5: History and polish

- Player match history
- Tournament history
- Organizer feedback views
- Accessibility and deployment hardening
